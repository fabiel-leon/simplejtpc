
comandos 
    -mail : enviar pag html con los vinculos cambiados a mailto
    -amp : igual que el anterior pero con el archivo adjunto para descargar y abrir con un visor de html
    -get : envia el archivo adjunto sin edicion ninguna( el arhivo tiene que tener un formato que appengine permita )
    -plain: enviar solo el texto de la pag web
    -zip : comprime el arhivo en zip sin  otra modificacion
    -zip-plain : extrae el texto de la pagina y luego la envia en zip
    -zip-mail : igual que el anterior pero envia la pag con vinculos mailto, para abrir directo desde el cliente de correo 
    -gzip: comprime el archivo en gzip sin otra modificacion 
    -gzip-plain : extrae el texto de la pag y lo comprime en gzip
    -gzip-mail : cambia los vinculos a mailto y comprime a gzip
